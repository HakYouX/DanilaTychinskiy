## 📂 Проекты

### [AI Telegram Bot](https://github.com/HakYouX/Telegramm-bot-AI)
*Умный Telegram-бот на базе библиотеки g4f с расширенным функционалом*
- **Задача:** Создать интеллектуального ассистента для решения разнообразных пользовательских запросов
- **Решение:** 
  - Интегрировал библиотеку g4f для AI-взаимодействия
  - Реализовал асинхронную обработку сообщений
  - Добавил систему кэширования и обработки ошибок
  - Настроил webhook для стабильной работы
- **Технологии:** Python, g4f, python-telegram-bot, asyncio, SQLite
- **Результат:** Бот обрабатывает 100+ запросов daily, время отклика зависит от вашего сервера

### [Торговый алгоритм для скальпинга](https://github.com/HakYouX/AI-Crypto-Trading-Terminal)
*Алгоритмическая торговая система для скальпинга на финансовых рынках*
- **Задача:** Разработать систему прогнозирования движения графика для автоматизации торговых решений
- **Решение:**
  - Реализовал анализ технических индикаторов (RSI, MACD, Moving Averages)
  - Создал алгоритм распознавания паттернов ценового действия
  - Настроил систему сигналов для точек входа/выхода
- **Технологии:** Python, pandas, numpy, matplotlib, ccxt (для биржевых API)
- **Результат:** Точность прогнозов 72% и выше, система генерирует сигналы в реальном времени
